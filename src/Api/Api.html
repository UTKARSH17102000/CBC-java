API - Application Programing interface It is intermediate layer between client and server where client asks some data using queries and goes to api which take data from the server and give back the data in the form of response

Application Programing interfaces allow services to communicate with each other
API allows developers to access data from a service without any knowladge of how codebase has been implemented

Methods - POST GET PUT DELETE
GET- Reading Information
POST - INSERT
DELETE - deleting data
PUT - UPDATE

HTTPS - The Schema of the request (Protocol)
